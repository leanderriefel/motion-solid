# Motion Solid Docs

Documentation for `motion-solid`.

## Getting Started

1. Install dependencies:

   ```bash
   bun install
   ```

2. Run development server:
   ```bash
   bun dev
   ```

## Structure

- `src/routes`: Pages and documentation (MDX supported).
- `src/components`: Components used in the docs (Layout, etc).
- `src/app.css`: Tailwind and global styles.

## Adding Docs

Create a new `.mdx` file in `src/routes/docs/`.
